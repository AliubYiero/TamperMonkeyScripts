# DouyinLiveClean

纯净化抖音直播间.

## 功能：

- **纯净化直播间**. 去除大部分广告和页面元素:
    - 默认窗口留下顶部的信息栏和搜索栏, 直播栏, 礼物栏和弹幕栏;
    - 网页全屏去除底部的礼物栏.
