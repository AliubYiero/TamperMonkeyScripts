# Style Kaoshibao Beautify

> 优化考试宝答题界面UI，增加按键绑定

## 功能

- 优化答题界面的UI
- 添加按键绑定, 可以通过键盘快速选择选项和跳转题目
- ~~模拟考试解锁随机选项功能~~, 该功能为 beta 功能, 不能保证其稳定性

### 按键绑定

- 选择选项：
    - `1`：选项A
    - `2`：选项B
    - `3`：选项C
    - `4`：选项D
    - `5 ~ 9`：选项E、F、G...
- 下一题：`Enter`、`+`、`→`
- 上一题：`-`、`←`
- 多选题提交：
    - 第一次`Enter`：提交答案
    - 第二次`Enter`：下一题
